🔢 What is SHA-256?

SHA-256 is a cryptographic function that produces a unique 64-character fingerprint of any file. Even a single changed pixel produces a completely different hash — making tampering mathematically detectable.

🌐 What is IPFS?

IPFS (InterPlanetary File System) stores files across a decentralised network. No single server controls your evidence — it cannot be deleted, suppressed, or tampered with by any authority.

⛓️ Blockchain timestamp

The SHA-256 hash is written to the Polygon blockchain with a timestamp. This creates a permanent, publicly verifiable record that this file existed in this exact state at this exact time — legally admissible.
